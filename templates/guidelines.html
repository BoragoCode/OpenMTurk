OpenMTurk Customizable image-labelling tool


On this demo site it's not possible to upload your own files for labelling, but we've included a couple of dummy images for you to test. Before you start labelling, please first read the below guidelines. In the box beneath the tool, you'll see the output of the label coordinates as you would get them in a json file.

Using the labelling tool


For each image, we currently aim to accurately gather:

  • The type of document that the picture contains.
  • The orientation of the document (is it sideways? (to the left, or to the right?). (is it upside-down? )
  • The location of headers in the image
  • The location of the text blocks in the image
  • The location of drawings or diagrams in the image
  • The location of tables in the image

Before you start labelling

Please follow these guidelines:

  • Make sure to submit your labels (by clicking the submit button or pressing the return key)
  • Labels must be very accurate. If you made a mistake, please clear your document from labels and re-label it.
